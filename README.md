# ULYXX3 — Portfolio BUT MMI 1

Site portfolio futuriste minimaliste réalisé en HTML, CSS et JavaScript.

---

## 📁 Structure du projet

```
Portfolio-zz/
├── index.html          ← Page d'accueil (point d'entrée)
├── pages/
│   ├── projets.html    ← Liste de tous les projets GitHub
│   ├── videos.html     ← Créations vidéo et animations
│   └── contact.html    ← Mes réseaux
├── assets/
│   ├── css/
│   │   └── style.css   ← Feuille de style (design system complet)
│   └── js/
│       └── script.js   ← JavaScript (injection données, filtres)
└── README.md
```

---

## 🚀 Lancer le site en local

Ouvrir `index.html` directement dans un navigateur, ou utiliser une extension type Live Server.

---

## ✏️ Personnaliser

- **Projets** : modifier le tableau `var projets = [...]` dans `assets/js/data.js`
- **Réseaux sociaux** : modifier le tableau `var socials = [...]` dans `assets/js/data.js`
- **Couleurs** : modifier les variables CSS dans `assets/css/style.css` (section `:root`)

---

## 🛠 Technologies

- HTML5 sémantique
- CSS3 (variables, grid, flexbox, animations)
- JavaScript ES5 (boucles, tableaux, fonctions, DOM)
